E-commerce Website Redesign - รายละเอียด

ประเภท: Web Design, UI/UX

บทบาท: Lead Designer, Front-End Developer (HTML/CSS)

คำอธิบาย: โปรเจกต์นี้เป็นการปรับโฉมเว็บไซต์ E-commerce เก่าให้ดูทันสมัยและใช้งานง่ายขึ้น เป้าหมายหลักคือการเพิ่มความน่าสนใจให้กับสินค้า และปรับปรุงประสบการณ์การช้อปปิ้งออนไลน์. ผมได้ออกแบบ Layout ใหม่, เลือกใช้โทนสีที่สะอาดตา, และปรับปรุง Flow การใช้งานตั้งแต่หน้าแรกไปจนถึงหน้า Checkout เพื่อให้ผู้ใช้สามารถค้นหาสินค้าและทำการสั่งซื้อได้อย่างรวดเร็วและง่ายดาย.

สิ่งท้าทาย: การทำให้เว็บไซต์ดูดีบนอุปกรณ์ทุกชนิด โดยเฉพาะการจัดการรูปภาพสินค้าจำนวนมากให้โหลดเร็วและแสดงผลได้ดีบนมือถือ.

ผลลัพธ์: หลังจากเปิดตัวดีไซน์ใหม่ เว็บไซต์ได้รับคำชมในด้านความสวยงามและการใช้งานที่ลื่นไหล ผู้ใช้งานใช้เวลานานขึ้นบนเว็บไซต์ และมีอัตรา Conversion ที่เพิ่มขึ้นอย่างเห็นได้ชัด.

Mobile App Landing Page - รายละเอียด

ประเภท: UI/UX, Web Design

บทบาท: UI/UX Designer

คำอธิบาย: โปรเจกต์นี้คือการออกแบบ Landing Page สำหรับโปรโมทแอปพลิเคชันมือถือใหม่ที่เน้นการจัดการ Tasks. ผมได้วิเคราะห์กลุ่มเป้าหมายและความต้องการของแอป เพื่อสร้างดีไซน์ที่น่าดึงดูดใจและสื่อสารคุณสมบัติหลักของแอปได้อย่างชัดเจน เน้นการใช้ภาพประกอบที่สวยงาม, ไอคอนที่เข้าใจง่าย, และ Call to Action ที่เชิญชวนให้ดาวน์โหลดแอป.

สิ่งท้าทาย: การนำเสนอข้อมูลที่ซับซ้อนของแอปให้กระชับและน่าสนใจภายในหน้าเดียว.

ผลลัพธ์: Landing Page นี้ช่วยเพิ่มยอดดาวน์โหลดแอปได้อย่างมีนัยสำคัญ และได้รับ feedback ที่ดีจากผู้ใช้.

Corporate Brand Identity - รายละเอียด

ประเภท: Branding, Web Design

บทบาท: Brand Designer, Web Designer

คำอธิบาย: โปรเจกต์นี้เป็นการสร้างสรรค์ Brand Identity ให้กับบริษัท Startup ด้านเทคโนโลยี ตั้งแต่โลโก้, โทนสี, ฟอนต์ ไปจนถึงคู่มือการใช้งาน Brand. จากนั้นผมได้นำ Brand Identity นี้มาพัฒนาเป็นเว็บไซต์องค์กรที่สะท้อนถึงค่านิยมและความเป็นมืออาชีพของบริษัท.

สิ่งท้าทาย: การสร้างแบรนด์ที่โดดเด่นและเป็นที่จดจำในตลาดที่มีการแข่งขันสูง.

ผลลัพธ์: บริษัทมี Brand Identity ที่แข็งแกร่งและเว็บไซต์ที่เป็นตัวแทนของบริษัทได้อย่างสมบูรณ์แบบ ช่วยสร้างความน่าเชื่อถือและดึงดูดนักลงทุน.
